# Shannon, Deng, Tsallis entropies and multi-modal bLSTM for ultrasound thyroid images classification
Clinicalrecords.xls includes data of 289 patients. Age, gender, body mass index (BMI), diagnosis of hypothyroidism or subclinical hypothyroidism, levothyroxine dose, smoking, history of neoplasm, first-degree family history of thyroid neoplasms, T4L levels, and TSH levels, diagnoses of prediabetes, diabetes, systemic arterial hypertension (HTN), dyslipidemia, osteoporosis, chronic kidney disease (CKD), and hyperparathyroidism can be found there. The label "benign" (TIRADS 1,2) of each record can be interpreted as no nodule was found to be malignant (TIRADS 3,4,5). When "malignant" is assigned, at least one nodule was found to be malignant. So each patient has several images with nodules. 
The folder images contain zipped images labeled as benign and malignant and can be identified by the patient ID in the name of the file. The pattern file name is "P0001....png" . For example, the file name "P0002 2024-07-17 a la(s) 10.15.47" is from the patient ID "P0002". The images and clinical records can be matched by ID_Clinicaldata of Clinicalrecords.xlsx
